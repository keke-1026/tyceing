# wince-typing
WindowsCE向けに作られたtypingゲームです｡
説明がめんどくさいので簡単なことしか説明しません｡
Creditはexe内に書いてあります｡
遊び方もexe内に書いてあります｡
本ソフトはOSSです｡収益化､改造等OKです｡
words.txtにて､出てくる単語を追加､削除できます｡構成は､「 [漢字等],[読み] 」 です｡
correct.wavは､打つ文字が正解の場合のSE､miss.wavはミスタイプの場合のSE､result.wavで終了時のSEが設定できます｡
また､なんか知らんがbgm.wavを入れると最初の画面で流れるはずですが流れないのでだれかうまく行ったらlssuesへGO
